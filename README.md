# Sales Performance Dashboard | Power BI

## Overview
This project is an interactive Sales Performance Dashboard developed using Power BI to analyze revenue, profit, customer trends, and regional sales performance. The dashboard helps businesses monitor key sales KPIs and make data-driven decisions.

## Objective
The objective of this project is to transform raw sales data into meaningful visual insights for tracking business growth, identifying top-performing regions, and improving decision-making.

## Key Features
- Revenue and profit analysis
- Region-wise sales performance tracking
- Customer and product analysis
- KPI cards for business metrics
- Interactive filters and slicers
- Trend analysis using visual dashboards
- Dynamic and user-friendly reporting
